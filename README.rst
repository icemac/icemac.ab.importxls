Hacking
=======

Source code
-----------

Get the source code::

   $ hg clone https://bitbucket.org/icemac/icemac.ab.importxls

or fork me on: https://bitbucket.org/icemac/icemac.ab.importxls

Running Tests
-------------

To run the tests yourself call::

  $ python2.7 bootstrap.py
  $ bin/buildout
  $ bin/py.test
